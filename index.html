FAQ

Comment changer les droits sur Linux ?

La commande chmod permet de modifier les droits et permissions. On utilise cette dernière en indiquant l'utilisateur (u, g, o ou a), les actions "+, -, =" et le droit concerné "r, w, x".

Que veut dire la propriété NaN en Js?

La propriété NaN signifie que la variable en question n'est pas un nombre (Not a Number)

Est-il possible de modifier un commit ?

La commande git commit --amend permet de modifier facilement le commit le plus récent.

Comment mettre à jour Ubuntu ?

La commande "sudo apt update" sert à chercher si des mises à jour sont disponibles, ensuite pour commencer la mise à jour du système, on utilise la commande "sudo apt upgrade".

Dois-je mettre la balise fin PHP ?> ?

La balise de fin de PHP n'est pas obligatoire, si les scripts sont destinés à être inclus ou requis, il est préférable de ne pas fermer la balise PHP pour moins de "problèmes".
